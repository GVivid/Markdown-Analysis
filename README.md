## Markdown-Analysis
To optimize a large Markdown repository, I made a program that generates canonical "hub" files. The system leverages semantic analysis to evaluate text similarity and maps document connections using wikilinks. By graphing these inter-file relationships, the application identifies content clusters, which can then be used to eliminate data redundancy and accelerate information retrieval.

This also gave me the opportunity to learn how to use a.i. more effectively and to learn about clustering.
